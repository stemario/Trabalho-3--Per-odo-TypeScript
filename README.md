PLATAFORMA DE TUTORIAIS
====================================================

1. DESCRIÇÃO DO PROJETO

A Plataforma de Tutoriais é uma aplicação desenvolvida
com o objetivo de auxiliar usuários no aprendizado de
tecnologias de programação por meio de uma interface
simples e intuitiva.

O projeto foi inicialmente criado em TypeScript para
execução no terminal e posteriormente adaptado para
uma interface web utilizando HTML e CSS, proporcionando
uma experiência mais amigável ao usuário.

====================================================
2. OBJETIVO

O objetivo deste projeto é permitir que o usuário
acesse conteúdos educacionais sobre tecnologia de forma
organizada, aplicando conceitos de programação e
desenvolvimento web.

====================================================
3. TECNOLOGIAS UTILIZADAS

- TypeScript
- JavaScript
- Node.js
- HTML5
- CSS3
- npm

====================================================
4. FUNCIONALIDADES

- Tela de boas-vindas.
- Personalização através do nome do usuário.
- Interface gráfica para navegação.
- Estrutura preparada para exibição de tutoriais.
- Organização do projeto utilizando TypeScript.

6. IMPORTÂNCIA DO HTML E CSS

HTML:
Responsável pela estrutura da página, criação de campos
de entrada, botões, títulos e áreas de conteúdo.

CSS:
Responsável pela aparência visual da aplicação,
definindo cores, espaçamentos, alinhamentos e layout.

A utilização dessas tecnologias permitiu transformar
uma aplicação de terminal em uma aplicação web acessível
através do navegador.

====================================================
7. COMO EXECUTAR O PROJETO

Caso deseje executar a versão TypeScript:

1. Compilar o código:

   npx tsc

2. Executar o arquivo compilado:

   node tutorial.js

----------------------------------------------------

Para visualizar a interface web:

1. Abrir o arquivo:

   plataforma-tutoriais.html

2. O navegador exibirá a página da plataforma.
